# Revenue-Driver-Analysis---Coffee-Shop

Analyzing the factors that influence Coffee Shop revenue.

Based on the analysis conducted using the Random Forest method on 2000 Coffee Shop customer data records, it can be concluded that the number of customers is the most influential factor affecting revenue. This is followed by the average purchase value and marketing expenses.

In addition, the Random Forest model achieved a high prediction accuracy, reaching 94.9%.

The selection of Random Forest as the prediction model in this analysis was baseed on its ability to capture complex relationships between variables, as it consists of multiple decisions trees. In addition, Random Forest is capable of measuring feature importance and predicting revenue with relatively high accuracy.
Random Forest is also considered robust againts outliers and does not require linearity assumptions like regression models.

The feature importance results generated by the Random Forest model are further supported by the importance values produced using SHAP.
SHAP is a method used to identify which features have the greates influence on the predictions made by a machine learning model.
